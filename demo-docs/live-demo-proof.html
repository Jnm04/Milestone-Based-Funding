2. System Architecture

The Cascrow platform implements a three-layer architecture:

  • Frontend: Next.js 15 application with MetaMask wallet integration for grant givers and receivers
  • Backend: Next.js API routes handling escrow creation, AI verification orchestration, and contract lifecycle management
  • On-chain: Solidity smart contracts on XRPL EVM Sidechain — MilestoneFundEscrow and MockRLUSD (testnet)

3. AI Verification Module

The AI verification module uses the Claude API (claude-sonnet model) to evaluate submitted proof documents against milestone criteria. The flow is as follows:

  • Receiver uploads a PDF proof document via the platform interface
  • Backend extracts text content from the PDF using pdf-parse
  • Extracted text is sent to Claude API with a structured system prompt containing the milestone criteria
  • Claude returns a structured JSON response with fields: decision (YES/NO), reasoning, and confidence (0–100)
  • If decision is YES and confidence ≥ 70, the escrow release transaction is automatically triggered
  • If decision is NO, the receiver is notified with the reasoning and may resubmit
